Collected information

The only data Insynsnotiser collects is what companies the user is currently subscribed to - This information is collected by Firebase Cloud Messaging.
Firebase Cloud Messaging registers an instance of a client app, obtaining a registration token that uniquely identifies the app instance, but not you. This data is used on the server to send notifications to right users.
No sensitive data is stored.
